# Random-Password-Generator
A simple and customizable random password generator built using HTML, CSS, and JavaScript. Users can generate secure passwords with options to include

• Uppercase letters
• Lowercase letters
• Numbers
• Symbols

Features

• Generate random secure passwords
• Select character types using checkboxes
• Easy-to-use interface
• Instant password display
• Lightweight and beginner-friendly project

Technologies Used

• HTML
• CSS
• JavaScript (DOM Manipulation)

How It Works

1. User selects password options:
   • Capital letters
   • Small letters
   • Numbers
   • Symbols
2. JavaScript creates a character pool based on selected options.
3. A random password is generated using Math.random().
4. Generated password is displayed on the screen.
